💎 钻石速刷攻略

1. 主线剧情

完成主线剧情任务是获取钻石的主要途径，每通过一章可获得大量钻石奖励。

2. 成就系统

完成各种成就目标可获得钻石奖励，包括角色培养、副本通关、竞技场排名等。

3. 竞技场奖励

每天根据竞技场排名发放钻石奖励，排名越高奖励越多。

4. 活动奖励

参与限时活动、节日活动可获得大量钻石奖励。

💰 金币速刷攻略

1. 金币副本

每日必刷金币副本，可获得大量金币奖励，难度越高奖励越多。

2. 主线副本

通关主线副本可获得金币奖励，重复刷已通关副本也可获取金币。

3. 出售装备

4. 日常任务

⚡ 体力获取攻略

1. 自然恢复

每5分钟恢复1点体力，体力上限随等级提升而增加。

2. 等级奖励

角色升级时会恢复一定量的体力。

3. 每日领取

每天固定时间可在活动页面领取体力奖励。

4. 钻石购买

每日可使用钻石购买体力3次，价格随购买次数递增。

5. 公会加成

加入公会可获得体力恢复速度加成。
